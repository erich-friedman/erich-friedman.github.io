Cover-Up Puzzles #2

For each set of 6 pentominoes:
a) Fit them into a 6×5 tray.
b) Divide them into 3 groups of 2, and make the same shape with each set.
c) Divide them into ANY 2 groups of 3, and make the same shape with each set. (10 puzzles total)
d) Pick ANY 2 pieces, and make the same shape as 2 other pieces. (15 puzzles total)
